# Business Moving Group Website

Modern Next.js 14 website for Business Moving Group. Commercial moving services in Orange County & LA.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- lucide-react icons
- Vercel deployment

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel --prod
```
